# Tic Tac Toe Game ❌⭕

A classic Tic Tac Toe game built with HTML, CSS, and JavaScript.
Play against a friend and see who wins!


##  Features

- 2 Player mode (Player X vs Player O)
- Winning combination detection
- Draw/Tie detection
- Score tracking for both players
- Restart/Reset button
- Clean and responsive UI

##  Built With

- HTML5
- CSS3
- Vanilla JavaScript

##  Getting Started

### Installation

1. Clone the repository:
```bash
   git clone https://github.com/nazibul906/tic-tac-toe.git
```

2. Navigate into the project folder:
```bash
   cd tic-tac-toe
```

3. Open `index.html` in your browser:
```bash
   # On Windows
   start index.html

   # On Mac/Linux
   open index.html
```

##  How to Play

1. The game is played on a **3x3 grid**
2. Player 1 is **X** and Player 2 is **O**
3. Players take turns clicking on empty squares
4. The first player to get **3 in a row** wins:
   - Horizontally ➡️
   - Vertically ⬇️
   - Diagonally ↗️
5. If all 9 squares are filled with no winner — it's a **Draw!**
6. Click **Restart** to play again
